#############
# Notice
#############

To store pointer to a constant object of a class
we must use vector of const pointer:

e.g.:
If our function of adding Genome pointer is:

void GenomeMatcherImpl::addGenome(const Genome& genome)
{
	genomes.push_back(&genome);
}

Our private vector of storing Genome* should be:
std::vector<const Genome*> genomes;
